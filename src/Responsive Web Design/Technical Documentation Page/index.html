Introduzione

CSS (Cascading Style Sheets) è un linguaggio utilizzato per descrivere la presentazione di documenti HTML.

Principali caratteristiche:

  • Separazione tra contenuto e presentazione
  • Supporto per vari dispositivi
  • Cascata e ereditarietà

CSS viene applicato attraverso regole che specificano come gli elementi devono essere visualizzati.

body { font-family: Arial, sans-serif; color: #333; }
Sintassi Base

Una regola CSS è composta da:

selettore { proprietà: valore; }

Esempio pratico:

h1 { color: blue; font-size: 24px; }

Ogni dichiarazione deve terminare con un punto e virgola.

Selettori

Tipi di selettori principali:

  • Elemento: div
  • Classe: .classe
  • ID: #id

I selettori possono essere combinati per maggiore specificità.

.container { width: 80%; margin: 0 auto; }
Box Model

Componenti principali:

div { width: 300px; padding: 20px; border: 1px solid #000; margin: 10px; }

Proprietà correlate:

  • box-sizing
  • display
  • position

Il box-sizing: border-box include padding e border nelle dimensioni totali.

Flexbox

Modello di layout per organizzare elementi complessi:

.container { display: flex; justify-content: center; align-items: center; }

Vantaggi principali:

  • Allineamento semplice
  • Distribuzione dello spazio
  • Layout responsive

Flexbox è particolarmente utile per layout dinamici.

.item { flex: 1; order: 2; }
